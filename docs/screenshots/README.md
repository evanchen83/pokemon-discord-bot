# Screenshots Guide

Drop screenshots in this folder and keep filenames simple with this pattern:

- `NN-area-short-title.png`
- Example: `01-discord-open-pack.png`

Use `NN` to control order in README/docs galleries (`01`, `02`, `03`, ...).

## Suggested filenames

- `01-discord-open-pack.png`
- `02-discord-my-cards.png`
- `03-discord-catch.png`
- `04-discord-pokebox.png`
- `05-discord-pokeagent.png`
- `06-grafana-overview.png`
- `07-grafana-top-sets.png`
- `08-grafana-command-outcomes.png`

## Optional notes file

If you want captions or section notes, edit:

- `docs/screenshots/manifest.md`

Keep each line in this format:

- `filename | caption`

Example:

- `01-discord-open-pack.png | Open pack command with paginated card embeds`

After you add images (and optional captions), tell me and I will format the README sections for you.
